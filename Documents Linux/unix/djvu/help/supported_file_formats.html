Supported file formats

DjVu Solo supports the encoding of the following file formats:

  • DjVu (*.djvu, *.djv)

  • BMP (*.bmp)

  • GIF (*.gif)

  • IW44 (*.iw44, *.iw4)

  • JPEG (*.jpeg, *.jpg)

  • PNM (*.pnm, *.ppm, *.pgm, *.pbm)

  • TIFF (*.tiff, *.tif)

  • PICT (*.pict, *.pct)

For information about the encoding process, see Opening and encoding an image file.
