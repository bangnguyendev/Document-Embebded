Changing display settings while viewing the document

If you want to change the display settings of a DjVu document while you are working with it, you can override the startup display settings. This does not affect the way documents appear when you open them in the DjVu Web Browser Plug-in.

To change display settings while you are working with a document:

  • On the View menu, click Display, and then choose the setting you want.

For more information about these settings, see the list in Setting startup display.
