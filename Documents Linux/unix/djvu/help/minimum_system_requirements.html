Minimum system requirements

The minimum system requirements for DjVu Solo 3.1 include the following:

  • 100 MHz processor with 32MB RAM; 266 MHz with 128MB RAM recommended

  • Microsoft® Windows® 95/98/2000 or Windows NT® 4.0 with Service Pack 6

  • CD-ROM drive

  • Mouse or other pointing device

  • Sufficient hard disk space for original images, output images, and temporary files created during the encoding process

  • DjVu Web Browser Plug-in (which you can download from the LizardTech Web site) to conduct word searches in DjVu documents
