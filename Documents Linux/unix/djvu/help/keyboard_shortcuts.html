Keyboard shortcuts

The following table lists the keyboard shortcuts you can use as an alternative to menu commands.

 

Press

To

Ctrl+S

Save a file.

Ctrl+P

Print a file.

Ctrl+F

View the first page of the document.

Ctrl+L

View the last page of the document.

Ctrl+N

View the next page.

Ctrl+R

View the previous page.

Ctrl+O

Open a file.

F2

Begin a slide show.

+ (plus sign)

Zoom in on the area of the image you click.

- (minus sign)

Zoom out of the image.
