Rearranging and removing pages

If you do not like the order of the pages in a multiple-page DjVu document, you can change the sequence at any time. If you decide you do not want to include a page you inserted, you can easily remove it from the document.

When you remove or rearrange pages in a DjVu document, page numbers change, which can make some hyperlinks point to incorrect locations. To prevent this, when you create hyperlinks, use page file names instead of page numbers.

To rearrange pages in a document:

  1. If thumbnails are not displayed, choose Thumbnails > Show Thumbnails.

  2. Select the thumbnail for each page you want to move.

TIP  To select a range of pages, hold down the Shift key while you select the thumbnails; to select one page at a time, hold down the Ctrl key.

  1. Right-click one of the thumbnails and choose Cut.

  2. Right-click the thumbnail for the page after which you want to insert the pages you are moving, and choose Paste After.

The thumbnails appear, showing new page numbers.

TIP  To quickly rearrange pages in a smaller document, drag thumbnails from one location to another.

To remove pages from a document:

  1. If thumbnails are not displayed, choose Thumbnails > Show Thumbnails.

  2. Select the thumbnail for each page you want to remove.

TIP  To select a range of pages, hold down the Shift key while you select the thumbnails; to select one page at a time, hold down the Ctrl key.

  1. Right-click one of the thumbnails and choose Delete Page(s) from the shortcut menu. Click Yes in the Confirm Page Deletion message that appears.

NOTE  You can also press the Delete key.
