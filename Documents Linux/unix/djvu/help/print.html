Printing DjVu documents

You can print a single page, a selected range of pages, or an entire document.

To print a page or document:

  1. Display the document you want to print, and then choose File > Print.

  2. In the Print dialog box, select the correct printer and number of copies you want to print.

  3. Under Print Range, select All to print the entire document or Pages to print a range of pages.

    If you select Pages, specify the page numbers of the first and last pages of the range.

  4. Select the zoom level you want: Default, Reduce to Fit, or Current Zoom.

    Default is 100% of the image size, Reduce to Fit reduces the image if it is larger than the paper size set for the printer, and Current Zoom is the magnification level at which the image currently appears.

  5. Click OK.
