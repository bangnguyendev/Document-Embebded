Editing and previewing documents

When you work with documents in DjVu Solo, the majority of your time is spent editing them. When you edit documents, they do not always appear the way they will to a user opening them in a browser. For example, when you add a hyperlink in Edit mode, you can see the handles that enable you to change the size of the linked area. However, you can easily preview the document the way your audience sees it, and then return to editing.

To preview a document:

  • Click the Preview mode button on the toolbar.

To edit a document:

  • Click the Edit mode button on the toolbar.

    NOTE  When you start DjVu Solo, it is set to Edit mode by default.
