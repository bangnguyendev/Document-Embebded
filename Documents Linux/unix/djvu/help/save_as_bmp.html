Saving documents as BMP files

You can save a document in BMP format if you want to open it in another bitmap-editing program to fine-tune it, or to distribute it to users who do not have access to the Web.

To save a file in BMP format:

  1. Click on the document (not the thumbnail) in the main DjVu Solo window.

  2. From the File menu, choose Export Page.

  3. In the Save As dialog box, type a name for the file, choose the location, and click Save.
