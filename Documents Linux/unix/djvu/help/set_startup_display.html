Setting startup display

You can determine which layers of a DjVu document or page display when the document is opened in the DjVu Web Browser Plug-in. DjVu Solo gives you the following options:

  • Color (Default)

    All image layers are displayed.

  • Black & white

    Only the black and white image layer is displayed, if present.

  • Foreground

    Only the image foreground layer is displayed, if present.

  • Background

    Only the image background layer is displayed, if present.

To change the startup display for a single page or an entire document:

  1. Display the page or document in the DjVu Solo window.

  2. For a single page, choose Edit > Page Startup Mode.

    For an entire document, choose Edit > Document Startup Mode. In the Startup Mode dialog box, select the display setting you want and click OK.
     

 
For additional information, see Changing display settings while viewing the document.
