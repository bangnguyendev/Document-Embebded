Setting page or document alignment

When the size of a DjVu image is smaller than the size of the DjVu Solo or browser window, you can set the position of the image within the window.

By default, the image is centered horizontally and vertically. In DjVu Solo, you can adjust the alignment in either direction.

 

This page has a horizontal alignment of Center and a vertical alignment of Top.

To change the page alignment:

  1. For a single page, display the page, and then choose Edit > Page Alignment.

    For the entire document, open the document, and then choose Edit > Document Alignment.

  2. In the Alignment dialog box, select the horizontal and vertical settings you want, and click OK.

    If you select Default, the image is aligned according to the browser’s settings, if applicable. Otherwise, the page is centered in both directions.
