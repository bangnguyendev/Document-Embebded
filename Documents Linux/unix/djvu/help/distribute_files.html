Components for distributing DjVu files electronically

When you distribute an indirect DjVu file electronically, you may need to include the following components along with the associated DjVu (.djvu or .djv) files:

  • Shared dictionary file (*.djbz)

The shared dictionary is a file that holds information that is repeated throughout the DjVu document. The DjVu document then, instead of repeating the information multiple times, contains a reference to the shared dictionary file where the information is needed.

Shared dictionary files are only created for some indirect DjVu documents. If no *.djbz files are present in the folder that contains your indirect document, you only need to distribute the associated DjVu files.
 

  • Thumbnail file (*.thumb)

    If you integrate thumbnails into your indirect DjVu document, the thumbnail information is stored in a file with a .thumb extension, which needs to be distributed with the associated DjVu files.

For more information about indirect DjVu documents, see Creating multiple-page DjVu documents.
