Setting zoom levels

You can set the level of magnification at which an image appears when users open the DjVu document in a browser. You can also change the zoom level while you are working on a document in DjVu Solo.

To set the startup zoom level:

  1. For an individual page, choose Edit > Page Startup Zoom.

    For an entire document, choose Edit > Document Startup Zoom.

  2. In the Startup Zoom dialog box, select the setting you want and click OK.

To change the zoom level while you are working in DjVu Solo:

  • From the View menu, choose Zoom, and then choose the level you want.

     

    TIP  To quickly change the zoom level, click the Zoom In or Zoom Out button on the toolbar, or press + (plus) to zoom in or - (minus) to zoom out.



The following list describes each zoom setting.

  • 300%, 150%, 100% (Default), 75%, 50%, 25%

    Select one of these settings to display the image at a percentage of the default size.

  • Fit Width

    Fits the page width in the DjVu Solo window. If you resize the DjVu Solo window, the page width adjusts to fit the window.

  • Fit Page

    Fits the entire page within the DjVu Solo window.

  • One To One

    Scales the image to display pixel-to-pixel. For this setting, each pixel of the original image corresponds to one pixel on the screen.

  • Stretch

    Sizes the page so that the entire image is always the same size as the browser window, regardless of how the user resizes the window.

  • Custom

    Opens the Custom Zoom dialog box, in which you can type a specific zoom level.
