Displaying page information

The Page information dialog box gives you the following statistics about the current page:

  • Image size (in pixels) and resolution

  • The amount of space allocated to certain portions of the page

  • The encoding ratio

To display page information:

  • From the View menu, choose Page Information.

or

  • Right-click on the document (not the thumbnail) in the main DjVu Solo window and select Page Information.
