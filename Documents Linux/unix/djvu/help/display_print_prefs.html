Setting display and printer preferences

You can use preference settings to customize the behavior of DjVu Solo.

To set preferences:

  1. From the Edit menu, choose Preferences to open the Preferences dialog box.

  2. Under Display Preferences, set the Screen color correction by dragging the slider until all squares in the grid look approximately the same.

  3. Under Printer Preferences, set the Printer color correction.

    By default, the Match screen colors box is selected, and the printer color is based on screen settings.
     
    To adjust the Printer color correction defaults, first clear the Match screen colors check box to the right, and then print a page on your printer. Determine how to change the setting according to the printed page, print again, and so on.

  1. Click OK.
