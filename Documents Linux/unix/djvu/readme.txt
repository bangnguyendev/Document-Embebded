	I Install DjVu Solo 3.1 (NC)
-------------------------------------------------------------------

To install DjVu Solo 3.1(NC) double-click on the DjVuSolo3.1_nc.exe file.

-------------------------------------------------------------------
	II Launch DjVu Solo 3.1 (NC)
-------------------------------------------------------------------
Launch DjVu Solo 3.1(NC) by selecting the START menu and navigating to 
Programs>LizardTech>DjVu Solo 3.1.

Alternately, the program may be launched directly from the 
C:\Program Files>LizardTech>DjVu Solo 3.1
    
-------------------------------------------------------------------
  	III Release Notes
-------------------------------------------------------------------


1. When selecting one to one zoom within a DjVu document the file
zooms to one pixel per dpi (not to 100%).

2. The startup default settings under EDIT menu on the toolbar  
do not change the settings when viewing the document within
Solo 3.1. The preferences are changed when viewing the document with the 
plug-in. When the file is launched using the Web Browser plug-in, 
it will display at the default setting.

3. To delete a hyperlink within a document, the right click button
on the mouse must be used.  

4. There is not an option to cut an image from another application
(i.e. the Web Browser Plug-in) and paste it into Solo 3.1 (NC).

5. PICT files with a resolution of more than 72 DPI won't open in Solo.
